浅谈css和媒体查询

用css来实现动画效果，如何实现呢？可以用伪类动画：hover。 hover 是指鼠标移入出现什么效果，它是立即发生，想要看到其中的变化过程，给他一个时间过度transition。before和after 简单举个用css画气球的例子：css部分：
.warp{ position:absolute;width:160px;height:160px;background:#785328;border-radius:160px 160px 60px 160px;-webki-transform:rotate(45deg);transform:rotate(45deg);box-shadow:-8px -8px 80px #848484 inset; }
.warp::after{ position:absolute;bottom:0;content:'';width:0;height:0;border:8px solid transparent;border-right-color:#848484;-webki-transform:rotate(45deg);transform:rotate(45deg); }

媒体查询是响适应的其中一种，它可以根据你设置的浏览器的宽度变化而重新渲染页面。它是由@media 控制。例如 @media (max-width: 900px) { }，这段代码的意思是，当你的浏览器窗口小于等于900px的时候，执行我这套css样式，从而达到重新渲染页面
